# MUSE_E – AI-Based Answer Script Evaluation System

MUSE_E is an AI-powered answer script evaluation system developed to automate the assessment process of descriptive answers using OCR, NLP, and semantic similarity techniques. The system is designed to reduce manual evaluation effort while improving consistency and efficiency in academic assessments.

---

## 🚀 Features

- Automated descriptive answer evaluation
- OCR-based text extraction from answer sheets
- Semantic similarity checking using NLP
- Role-based authentication system
- Secure data handling
- User-friendly interface
- Answer score prediction and evaluation

## 🔄 Workflow

1. User uploads answer script
2. OCR extracts text from the document
3. NLP preprocessing is performed
4. Answers are compared with reference answers
5. Cosine similarity calculates semantic relevance
6. Scores are generated automatically
